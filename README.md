# 🎓 Java Academic Projects - UoM

Αυτό το αποθετήριο (repository) περιέχει συγκεντρωμένο όλο το υλικό, τις ασκήσεις και τα παραδείγματα κώδικα από τα μαθήματα προγραμματισμού σε **Java** του **Πανεπιστημίου Μακεδονίας (UoM)**.

## 📁 Δομή του Repository

Ο κώδικας είναι οργανωμένος ανά θεματική ενότητα, ακολουθώντας την ύλη των μαθημάτων:

* **`Intro-to-Java/`**: Βασικές αρχές, μεταβλητές, βρόχοι (loops) και πίνακες.
* **`Object-Oriented-Programming/`**: Κλάσεις, Αντικείμενα, Κληρονομικότητα, Πολυμορφισμός και Interfaces.
* **`Data-Structures/`**: Υλοποίηση δομών όπως Lists, Stacks, Queues και Trees.
* **`Software-Engineering/`**: Design Patterns και αρχές καθαρού κώδικα (SOLID).
* **`GUI-Swing-JavaFX/`**: Δημιουργία γραφικών διεπαφών.

## 🚀 Τεχνολογίες & Εργαλεία

* **Γλώσσα:** Java (Version 17+)
* **Build Tool:** Maven / Gradle (Προαιρετικά)
* **IDE:** IntelliJ IDEA, Eclipse, ή VS Code
